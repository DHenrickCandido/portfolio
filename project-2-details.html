Weekly Spotify Wrapped ETL

Project information

  • Category: Data
  • Type: Personal Project

I'm building an ETL pipeline using Airflow to extract data from Spotify API, store it in Google Cloud Storage, process and transform it with Dataproc, and send it to BigQuery for analysis and Data Studio for visualization.


Additionally, I'm creating a separate pipeline that will send me weekly emails with insights from the data of the songs I listened to in the past week. Although the primary goal of this project is to consolidate my knowledge of GCP tools, it will also give me insights into my music preferences that I can use to discover new music and understand my listening habits better.
